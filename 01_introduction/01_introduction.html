GSAPとは

GSAP（GreenSock Animation Platform）は、Webサイト向けの高機能なJavaScriptアニメーションライブラリです。 複雑な動きをシンプルなコードで実現でき、滑らかでパフォーマンスの高いアニメーションを作成できます。

GSAPの主な特徴

  • DOM要素のアニメーション: 位置、サイズ、色、透明度などのCSSプロパティを簡単にアニメーション化
  • SVG要素のアニメーション: ベクター形式の画像や図形も自在に動かせる
  • 高いパフォーマンス: 最適化されたコードで滑らかなアニメーションを実現
  • ブラウザ互換性: モダンブラウザから古いブラウザまで幅広く対応
  • 充実したAPI: アニメーションの制御（一時停止、再開、逆再生など）が可能
  • プラグイン拡張: ScrollTriggerなど、様々な機能をプラグインで拡張できる

GSAPでできること

GSAPを使用することで、以下のようなアニメーションを簡単に実装できます：

  • 要素の移動、回転、拡大・縮小
  • 色の変化やフェードイン・アウト
  • 複数のアニメーションを連続して実行（タイムライン機能）
  • スクロールに連動したアニメーション（ScrollTriggerプラグイン）

GSAPの基本概念

トゥイーン（Tween）

GSAPの中心的な概念で、ある状態から別の状態への変化を定義します。 「Aの状態からBの状態へn秒かけて変化させる」という指示をコードで表現します。

タイムライン（Timeline）

複数のトゥイーンを順序立てて管理する機能です。アニメーションのシーケンスを簡単に作成できます。

イージング（Easing）

アニメーションの加速・減速パターンを定義します。自然な動きを表現するために重要な要素です。

実際のコード例

以下は、要素を右に100px移動させる最も基本的なGSAPコードです：

// 要素を1秒かけて右に100px移動
gsap.to(".my-element", {
  x: 100,
  duration: 1,
  ease: "power1.out"
});

主要なGSAPメソッド

// 要素を特定の状態に変化させる
gsap.to(".element", {プロパティ});

// 要素を現在の状態から開始し、指定した状態から現在の状態へ変化させる
gsap.from(".element", {プロパティ});

// 開始状態と終了状態の両方を明示的に指定
gsap.fromTo(".element", {開始プロパティ}, {終了プロパティ});

// タイムラインの作成
const tl = gsap.timeline({オプション});
tl.to(...).from(...).fromTo(...);
