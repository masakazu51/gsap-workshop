このリポジトリは、GSAP（GreenSock Animation Platform）の勉強会で使用する資料集です。

勉強会の概要

  • 対象: Web アニメーションの初学者
  • 目的: GSAP の基本を理解し、実際にアニメーションを実装できるようになる
  • 時間: 約 1 時間

資料の構成

  1. GSAP の概要 - GSAP とは何か、何ができるのか
  2. GSAP のメリット - 他のアニメーションライブラリとの比較
  3. Web での良いアニメーション - 自然な動きとタイミング
  4. GSAP の基本構文と例 - 基本構文と実際に動くシンプルな例
  5. 実践的なアニメーション例:
    • ボタンのホバーアニメーション
    • タイムラインシーケンス
    • スクロールフェードイン
    • スクロール連動アニメーション
    • スタッガーアニメーション

基本的なGSAPコード例

// 基本的なアニメーション
gsap.to(".element", {
  x: 100,
  y: 50,
  rotation: 45,
  opacity: 0.5,
  duration: 1,
  ease: "power2.inOut"
});

// タイムライン
const tl = gsap.timeline({defaults: {duration: 0.5}});
tl.to(".box1", {x: 200})
  .to(".box2", {y: 100}, "-=0.3")
  .from(".box3", {opacity: 0});

// スクロールトリガー
gsap.from(".fade-in", {
  scrollTrigger: {
    trigger: ".fade-in",
    start: "top 80%",
    end: "bottom 20%",
    toggleActions: "play none none reverse"
  },
  opacity: 0,
  y: 50
});

勉強会の進め方

各セクションを順番に進み、実際にコードを書きながら理解を深めていきます。セクション 5 の実演では、実際にブラウザで動作するサンプルを一緒に見ながら学びます。
