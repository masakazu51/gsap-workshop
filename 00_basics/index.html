1. 基本的な移動（gsap.to）

要素を右に移動させる基本的なアニメーション

// 要素を右に100px移動
gsap.to("#box1", {
  x: 300,
  duration: 2
});

2. 初期状態からのアニメーション（gsap.from）

右側から現在位置へ移動してくるアニメーション

// 右側から現在位置へアニメーション
gsap.from("#box2", {
  x: 300,
  duration: 2
});

3. 回転アニメーション

要素を回転させるアニメーション

// 要素を360度回転
gsap.to("#box3", {
  rotation: 360,
  duration: 2
});

4. 複数プロパティのアニメーション

位置、回転、大きさ、透明度を同時にアニメーション

// 複数のプロパティをアニメーション
gsap.to("#box4", {
  x: 300,
  rotation: 360,
  scale: 1.5,
  opacity: 0.5,
  duration: 2
});

5. repeat と yoyo

アニメーションを繰り返したり、往復させたりする

// 繰り返し（3回）
gsap.to("#box5", {
  x: 300,
  duration: 1,
  repeat: 2  // 合計3回実行される
});

// 往復（3回）
gsap.to("#box5", {
  x: 300,
  duration: 1,
  repeat: 2,
  yoyo: true  // 往復する
});

6. 異なるイージング

イージング（加減速パターン）の違いを比較

// 直線的（一定速度）
gsap.to("#box6", {
  x: 300,
  duration: 2,
  ease: "none"
});

// 減速
gsap.to("#box6", {
  x: 300,
  duration: 2,
  ease: "power1.out"
});

// 加速
gsap.to("#box6", {
  x: 300,
  duration: 2,
  ease: "power1.in"
});

// 加速後減速
gsap.to("#box6", {
  x: 300,
  duration: 2,
  ease: "power1.inOut"
});

7. stagger効果（複数要素の時間差）

複数の要素を時間差でアニメーションさせる

// 複数要素を時間差でアニメーション
gsap.to(".stagger-box", {
  y: 50,
  opacity: 0.5,
  duration: 1,
  stagger: 0.2  // 0.2秒ずつ遅れて開始
});
