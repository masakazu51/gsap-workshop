GSAPを使ったボタンホバーエフェクト

ユーザーがボタンにカーソルを合わせたときにアニメーションを実行することで、インタラクティブ性を高めることができます。このページでは、GSAPを使ったボタンホバーエフェクトのサンプルを紹介します。

基本的なホバーエフェクト

マウスを乗せると拡大し、マウスを外すと元のサイズに戻るシンプルなエフェクトです。

// 基本的なホバーエフェクト
const scaleButton = document.querySelector(".hover-scale");

// マウスが乗ったとき
scaleButton.addEventListener("mouseenter", () => {
  gsap.to(scaleButton, {
    scale: 1.1,
    duration: 0.3
  });
});

// マウスが外れたとき
scaleButton.addEventListener("mouseleave", () => {
  gsap.to(scaleButton, {
    scale: 1,
    duration: 0.3
  });
});

バウンスエフェクト

マウスを乗せるとバウンドするアニメーションを追加したボタンです。イージング関数を使って跳ねるような動きを表現しています。

// バウンスエフェクト
const bounceButton = document.querySelector(".hover-bounce");

// マウスが乗ったとき
bounceButton.addEventListener("mouseenter", () => {
  gsap.to(bounceButton, {
    scale: 1.1,
    y: -10,
    duration: 0.4,
    ease: "back.out(1.7)" // バウンスするイージング
  });
});

// マウスが外れたとき
bounceButton.addEventListener("mouseleave", () => {
  gsap.to(bounceButton, {
    scale: 1,
    y: 0,
    duration: 0.3,
    ease: "power2.out"
  });
});

複合エフェクト

回転と色の変化を組み合わせたエフェクトです。複数のプロパティを同時にアニメーションさせることができます。

// 複合エフェクト
const complexButton = document.querySelector(".hover-complex");

// マウスが乗ったとき
complexButton.addEventListener("mouseenter", () => {
  gsap.to(complexButton, {
    backgroundColor: "#e74c3c",
    rotation: 5,
    scale: 1.05,
    duration: 0.3
  });
});

// マウスが外れたとき
complexButton.addEventListener("mouseleave", () => {
  gsap.to(complexButton, {
    backgroundColor: "#3498db",
    rotation: 0,
    scale: 1,
    duration: 0.3
  });
});

まとめ

GSAPを使うことで、CSSだけでは難しい複雑なアニメーションを簡単に実装できます。特にイージング関数を活用することで、より自然で洗練された動きを実現できます。

上記のサンプルを組み合わせたり、他のプロパティを追加したりして、オリジナルのホバーエフェクトを作ってみましょう。
