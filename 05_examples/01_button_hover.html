GSAPでボタンホバーを実装する

CSSのtransitionだけでは実現できない複雑なホバーエフェクトをGSAPを使って簡単に実装できます。 このページでは、3つのボタンホバーエフェクトの例を紹介します。

1. 基本的なスケールエフェクト

マウスオーバー時に少し拡大するシンプルなエフェクトです。

実装コード:

// ボタン要素の取得
const button = document.getElementById("basic-button");

// マウスが入ったときのアニメーション
button.addEventListener("mouseenter", () => {
  gsap.to(button, {
    scale: 1.1,  // 10%拡大
    duration: 0.3,
    ease: "power1.out"
  });
});

// マウスが出たときのアニメーション
button.addEventListener("mouseleave", () => {
  gsap.to(button, {
    scale: 1,  // 元のサイズに戻る
    duration: 0.3,
    ease: "power1.in"
  });
});

2. バウンスエフェクト

マウスオーバー時にバウンスするエフェクトです。弾むような動きが特徴です。

実装コード:

// ボタン要素の取得
const bounceButton = document.getElementById("bounce-button");

// マウスが入ったときのアニメーション
bounceButton.addEventListener("mouseenter", () => {
  gsap.to(bounceButton, {
    scale: 1.15,  // 15%拡大
    duration: 0.4,
    ease: "elastic.out(1, 0.3)"  // 弾むようなイージング
  });
});

// マウスが出たときのアニメーション
bounceButton.addEventListener("mouseleave", () => {
  gsap.to(bounceButton, {
    scale: 1,  // 元のサイズに戻る
    duration: 0.2,
    ease: "power1.in"
  });
});

3. 複合エフェクト

回転と色の変化を組み合わせた複雑なエフェクトです。複数のプロパティを同時にアニメーションできます。

実装コード:

// ボタン要素の取得
const complexButton = document.getElementById("complex-button");

// マウスが入ったときのアニメーション
complexButton.addEventListener("mouseenter", () => {
  gsap.to(complexButton, {
    scale: 1.05,          // 少し拡大
    rotation: 5,          // 少し回転
    backgroundColor: "#e74c3c",  // 色を変更
    boxShadow: "0 5px 15px rgba(0,0,0,0.2)",  // 影を付ける
    duration: 0.3,
    ease: "back.out(1.7)"  // 少し行き過ぎてから戻るイージング
  });
});

// マウスが出たときのアニメーション
complexButton.addEventListener("mouseleave", () => {
  gsap.to(complexButton, {
    scale: 1,             // 元のサイズに戻る
    rotation: 0,          // 回転をリセット
    backgroundColor: "#3498db",  // 元の色に戻す
    boxShadow: "none",    // 影を消す
    duration: 0.3,
    ease: "power2.out"
  });
});

まとめ

GSAPを使うことで、CSSだけでは実現が難しい複雑なアニメーションも簡単に作成できます。 さまざまなイージング関数や複数プロパティの同時アニメーションを活用して、 ユーザー体験を向上させるインタラクションを実装しましょう。

これらのサンプルを参考に、独自のホバーエフェクトを作成してみてください。
